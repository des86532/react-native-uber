# react-native-uber

1. add an `.env` file
```
GOOGLE_MAPS_APIKEY='your own google api key'
```

2. install node_modules
```
npm install
```

3. install expo cli global (if you have installed, you can skip this step and go to step 4)
```
npm install --global expo-cli
```

4. start expo
```
expo start
```

5. choose device
